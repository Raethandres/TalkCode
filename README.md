#  **Talkcode**
### Web Application to write questions, answers, create your snippets and share


---   

Packages Required

* flask>=0.10
* flask_mysqldb
* flask_sqlalchemy
* sqlalchemy
* sqlalchemy_utils
* flask_script
* flask_migrate
* flask-login
* alembic
* pycountry
* flask_wtf
* wtforms_components
* wtforms

Install packages

```python
python setup.py install

```
